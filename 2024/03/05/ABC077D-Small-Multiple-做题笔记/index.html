ABC077D Small Multiple 做题笔记

发表于 2024-03-05

题意：给定正整数 $k$，并定义 $f(x) = x$ 的数字和，求一个正整数 $n$ 使得 $f(n\times k)$ 最小，并输出 $f(n\times k)$（$2 \le k \le 10^5$）。

显然一个朴素的想法是枚举 $n$，然后愉快地 T 飞。于是会想到猜各种结论以优化这一过程，然后就会发现很难做。

然而这样子做是没有出路的，因为本题需要从数和数字和入手。考虑任一正整数 $X$，它必然可以从数 $1$ 通过以下两种方式变换而来：

  • $x \gets x \times10$

  • $x \gets x + 1$

（其中 $x$ 的初始值为 $1$。）

然后我们观察这两种变换中 $x$ 的数字和发生了怎样的变化：在第一种变化中，相当于在 $x$ 的后面添加了一个 $0$，数字和是不变的；而在第二种变化中，在不考虑进位的前提下，相当于是给 $x$ 的数字和也加了 $1$。

于是我们可以将原题转化为下述形式：现有 $x = 1$，每次可以花费 $0$ 的代价使得 $x \gets x \times 10$，或者花费 $1$ 的代价使得 $x \gets x + 1$；求一个最小代价，使得变换之后的 $x \equiv 0 \mod k$。

显然针对这个问题，我们可以对于满足 $0 < x < k$ 的任一 $x$ 向 $x \times10 \ \text{mod}\ k$ 连一条边权为 $0$ 的边，向 $(x + 1) \ \text{mod}\ k$ 连一条边权为 $1$ 的边，然后从 $1$ 到 $0$ 跑最短路。$0/1$ BFS 即可。

但是还需要注意，在前面的描述中，令 $x \gets x + 1$ 需支付 $1$ 代价的前提是不发生进位，但显然这一条件在我们连边的过程中没有被保证。然而这是不用被考虑的，因为如果走的 $1$ 边已经多到了发生进位了，那么它一定不是最短路，读者自证不难。

其实这里应该也可以连边的时候处理一下，但是会慢一点，并且不好写。

以及我的 $0/1$ BFS 因为一些奇怪的原因挂掉了。
