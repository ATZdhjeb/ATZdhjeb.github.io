P1852 跳跳棋 做题笔记

发表于 2024-03-05

题意：感觉原题面没有人读不懂。

应该是很经典的 LCA 题了。以下我们钦定 $a < b < c,x < y < z$。

先对 $(a,b,c)$ 进行研究，发现无论是怎样的 $(a,b,c)$，均可以以 $a$ 或 $c$ 为轴把 $b$ “向外跳”。但是如果以 $b$ 为轴“向内跳”，就不是 $a$ 和 $c$ 都可以了。如果 $b - a < c - b$，则 $c$ 不能向内跳，否则它就会到 $a$ 的左边去，与题意不符；同理，若 $b - a > c - b$，则 $a$ 不能向内跳。所以，对于大部分 $(a,b,c)$，均有三种转移方式；但特殊地，$b - a = c - b$ 时，这个状态便只能让 $b$ 向外跳，因为只要向内跳，则 $a,c$ 必定重合。

然后就是逆天思维了：

注意到上述转移方式与二叉树类似，所以我们把每个 $(a,b,c)$ 视为一个节点，以 $a$ 为轴向外跳则转移至其左儿子，以 $b$ 为轴则转移至右儿子；可以发现，向内跳就是转移到父亲。而 $b - a = c - b$ 的节点因为不能向内跳，就是根。

剩下的就是在这棵树上求起始位置和终止位置的 LCA 了。当然，如果两个位置根本就不在一棵树上，那就输出无解。但是这棵树无限延伸，并且数据范围是 $10^9$，意味着不能用传统的求 LCA 方式。但有一点不变：求 LCA 的过程中，只需要用到一种操作，即走到父亲，即向内跳。

考虑 $(a_0,b_0,c_0)$，钦定 $b_0 - a_0 > c_0 - b_0$，那么它的父亲 $(a_1,b_1,c_1)=(a_0,2b_0-c_0,b_0)$。此时若 $b_1-a_1>c_1-b_1$，那么它还可以跳到父亲 $(a_2,b_2,c_2)=(a_0,3b_0-2c_0,2b_0-c_0)$。一直到 $(a_n,b_n,c_n)$，使得 $b_n-a_n\le c_n-b_n$。令 $d_0 = b_0 -a_0,D_0=c_0-b_0$，那么这一操作本质上就是保持 $a$ 不动，并令 $(d_0,D_0)\gets(d_0\ \text{mod}\ D_0,D_0)$，容易发现是一轮 $\gcd$。所以可以用 $\gcd$ 的方式来加速跳父亲的过程。

接下来再想怎么求 LCA。用倍增求 LCA 的思想，先把两个状态跳到同一深度（求深度可以二分跳多少步恰好到根），然后再二分一个步数，两侧同时跳该步数，若跳到同一点则减少跳的步数，反之增加。

然后就做完了。时间复杂度大约是 $O(\log^2)$。注意一下细节。
